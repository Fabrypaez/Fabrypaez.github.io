What kind of information can the user control and change?

In both applications the user can change the language, make adjustments to the audio, control notifications but, in Spotify you can change list' visualization from icons to list and vice versa.

Is the interface "inductive" or "deductive"?

Both applications use an inductive interface because they are easy-to-use applications that comply with the following guidelines:

One task per screen, easy screen controls, obvious next step, consistent screen template and progressive disclosure.
