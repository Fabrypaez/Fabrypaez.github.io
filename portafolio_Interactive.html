Interactive Body Installation.

Transfiguration Interactive Body Installation is the result of exploring digital sensors, generative art and motion capture, emphasising their interaction with the body and digital tech. This interactive installation presents an abstract and personal inner journey when each element allows the body to express itself in a natural state within a constant game with technology and digital sensors.

An interactive installation is a fun way to bring technology, sound, and light into a physical setting that responds to the user's interaction. They are designed in various forms, but they always employ a wide range of sensors to let users play with engaging and fascinating interactive digital content.

Developing Transfiguration Interactive Body Installation.

The journey to create Transfiguration began with the curiosity about motion capture; as we see before, this technique has evolved over the years and can be used in different aspects of our lives. Following the next logic: devices need to communicate, sensors communicate to microcontrollers, microcontrollers to computers, and computers to the Internet, the search for hardware and software began.

Nowadays, we can find many options regarding software and hardware that can deal with motion capture, from a simple web camera to sophisticated motion sensors. This research came across: Arduino, which is an open-source electronics platform based on an easy way to use hardware and software; Arduino uses serial communication, which led us to use software that received this information ready to transform it into graphics. Following the line of open source software, p5*js and Processing are the most popular software in creative coding that reunite the requirements for the interactive installation.

Transfiguration Interactive Body Installation uses a video projector as the output for the data that is being captured through our input devices. Its interaction with the viewer begins when it approaches the installation. At first, their image is captured by the webcam creating a digital reflection and projected onto the canvas. Then when the persona is in a range of 1.5-2 meters, the body movement is being registered by the ultrasonic sensor, which transforms the shown graphics giving them an extraordinary kinaesthetic experience.

For example, using sensors in performance allows exploring the relationships with our bodies and how we move them. Sensory technologies will enable us to experience or transcend our bodies and how their study can be expanded in virtual space, like a view of the body as language or data that can be read and manipulated. This project explores the concepts of how our body behaves when its image and movement are being captured. The interaction with this code is similar to a surveillance camera with a motion sensor. However, this code is not designed to spy on the user or collect his data; it is designed so that the user can play and explore as the body in motion alters the video.
