  • Interactive Art,

About

This sketch is designed to be used by VJ (Visual Jockey) or DJ (Disc Jockey) in stages with a MIDI (KORG nanoKontrol2) console. Allowing them to play with the lines' colours, control fade in/out of the image, and even save an image.

The Code as Creative Medium book prompt that I used in this project is"drawing lines." There are many examples with lines and interactivity, and in my sketch, I used two pseudocodes to design the graphic. I used the P5.Js example "Bouncing Line" from the website Happy Coding and for the interactivity with the MIDI console, I used the code "MIDI slider Demo" by Golan from the P5.Js platform.

Sketch

Have fun but you will need to have a MIDI KORG nanoKontrol2 console.
